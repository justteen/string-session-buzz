## Configs:

- API_HASH

  - Get from [Here](https://my.telegram.org).

- API_ID

  - Get from [Here](https://my.telegram.org).

- API_KEY

  - Heroku API Key from [Here](https://dashboard.heroku.com/account).

- APP_NAME

  - Heroku App Name.

- BOT_TOKEN

  - Telegram Bot Token from [here](https://t.me/BotFather).
